# Automatic Irrigation System using 8051 Microcontroller

## Project Overview

This project is an 8051 microcontroller-based automatic irrigation system that automatically controls water supply based on soil moisture levels.

When the soil is dry, the pump is turned ON. When the soil has sufficient moisture, the pump is turned OFF.

---

## Components Used

- 8051 Microcontroller
- Soil Moisture Sensor
- Relay Module
- Water Pump
- Power Supply
- Connecting Wires

---

## Working Principle

The soil moisture sensor continuously checks the moisture level of the soil.

- Dry soil → Water pump ON  
- Wet soil → Water pump OFF  

The 8051 microcontroller processes the sensor data and controls the relay accordingly.
