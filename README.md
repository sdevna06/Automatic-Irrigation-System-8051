# Automatic Irrigation System using 8051 Microcontroller

## Project Overview

This project is an 8051 microcontroller-based automatic irrigation system that monitors soil moisture and controls water supply automatically.

When the soil is dry, the system turns ON the pump. When moisture is sufficient, the pump is turned OFF.

---

## Components Used

- 8051 Microcontroller
- Soil Moisture Sensor
- Relay Module
- Water Pump
- Power Supply
- Breadboard & Wires

---

## Working Principle

The soil moisture sensor continuously checks soil condition.

- Dry soil → Pump ON  
- Wet soil → Pump OFF  

The 8051 microcontroller processes the sensor input and controls the relay accordingly.

---

## Features

- Automatic watering system
- Water conservation
- Low-cost embedded solution
- Real-time soil monitoring
